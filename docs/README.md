# 📔 Diario de Abigail - Documentación Principal

Bienvenido a la documentación oficial del **Diario de Abigail (v1.0.0)**. Este repositorio contiene una aplicación web progresiva diseñada para el crecimiento personal, la reflexión espiritual y el seguimiento de hábitos.

2.  **[Especificaciones de Diseño (UI/UX)](./DESIGN.md)**: Guía de estilos, colores, tipografía y tokens de diseño para diseñadores.
3.  **[Guía del Desarrollador](./DEVELOPER_GUIDE.md)**: Instrucciones para instalar, ejecutar, depurar y extender el código.

5.  **[Manual de Usuario](./USER_MANUAL.md)**: Guía de funcionalidades para el usuario final (Abigail).
