# 🧪 Documentación de Pruebas

## 1. Estrategia de Testing
En la versión v1.0.0, la estrategia se centra en Pruebas Manuales de Aceptación (UAT) y pruebas de persistencia.

## 2. Casos de Prueba (Checklist)
- **CP-01:** Al marcar una tarea y refrescar el navegador, la tarea debe seguir marcada.
- **CP-02:** El menú móvil debe responder correctamente a los toques y cambiar la vista.
- **CP-03:** El contador de hábitos debe incrementarse solo cuando se marca un checkbox.
- **CP-04:** La animación de GSAP no debe causar "flickering" en el renderizado inicial.

## 3. Herramientas de Depuración
- **React Developer Tools:** Para inspeccionar el estado del Contexto.
- **Chrome Storage Tab:** Para verificar la integridad del JSON en LocalStorage.
