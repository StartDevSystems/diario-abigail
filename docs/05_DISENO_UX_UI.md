# 🎨 Documentación de Diseño (UX/UI)

## 1. Filosofía de Diseño
El diseño busca transmitir paz, orden y delicadeza. Inspirado en el concepto de "Scrapbooking" digital.

## 2. Guía de Estilos (Style Guide)
- **Colores Primarios:** Rose Pastel (#fdf2f2), Cream (#fffcf2).
- **Colores de Contraste:** Deep Rose (#be185d), Soft Grey (#374151).
- **Radios de Borde:** 40px (`rounded-[2.5rem]`) para suavizar la estructura.

## 3. UX (Experiencia de Usuario)
- **Micro-interacciones:** GSAP suaviza la entrada de los paneles laterales.
- **Feedback Visual:** Checkboxes con animaciones de escalado al ser marcados.
- **Jerarquía:** Los títulos en Serif (Playfair Display) guían el ojo hacia las secciones importantes.

## 4. Diseño Responsive
- **Breakpoint Móvil (<768px):** Transición automática a Bottom Navigation Bar. El Sidebar se oculta completamente.
- **Efectos:** Glassmorphism (`backdrop-blur`) para mantener la cohesión visual sobre el fondo 3D.
