# 🔗 Documentación de Integración

## 1. Puntos de Extensión
- **Exportación de Datos:** El sistema permite el acceso programático al `localStorage` para exportar el JSON del diario.
- **Hooks de Terceros:** Compatible con herramientas de analítica (Google Analytics) mediante la inserción de scripts en `layout.tsx`.

## 2. Webhooks y Eventos
Actualmente no implementados. Se planean para notificaciones push en la v2.0.

## 3. Integración con Servicios de Versículos
Se ha dejado preparada la interfaz `Devotional` para conectar con APIs externas (Bible Gateway API) en futuras iteraciones.
