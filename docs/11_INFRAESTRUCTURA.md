# ☁️ Documentación de Infraestructura

## 1. Modelo de Servicio
SaaS (Software as a Service) basado en arquitectura Serverless provista por Vercel.

## 2. Recursos Utilizados
- **Vercel Edge Network:** Para la distribución global de los archivos estáticos.
- **Next.js Image Optimization:** Para el manejo eficiente de cualquier asset visual futuro.
- **HTTPS/SSL:** Certificados automáticos gestionados por el proveedor de hosting.

## 3. Escalabilidad
La infraestructura es elásticamente escalable. No requiere intervención manual para manejar picos de tráfico.
